# Game Plan

## コンセプト

別れではなく「また会う約束」を描く、太平洋横断の短い旅。

## 主人公

主人公はジャック。

TONTEKOが操縦する飛行機に乗り、アメリカから日本へ向かう。
旅の途中で、日本で過ごした時間やお店で出会った人たちとの思い出を集める。

## 画面構成

1. タイトル画面
2. アメリカ出発シーン
3. 太平洋を進む横スクロール収集ステージ
4. 日本到着シーン
5. 集めた思い出のコレクション
6. 映画風エンディングポスター
7. メッセージとスタッフロール

## 操作

- PC: 上下矢印キー
- スマホ: タップした位置へ飛行機が上下移動
- 共通: 音声ON/OFF

## ゲーム時間の目安

- タイトル・説明: 15秒
- 飛行ステージ: 約90秒
- コレクション・到着演出: 約25秒
- エンディング・スタッフロール: 約40秒
- 合計: 約2分50秒

## デザインメモ

- 温かい夕暮れ色を中心にする
- 映画ポスターのような大きな英字タイトル
- 可愛さより、親しみと少しのノスタルジーを優先
- 操作ボタンや説明は英語圏ユーザーが迷わない短い表現にする
- スマホでは縦持ち・横持ちの両方で主要要素が見えるようにする

## 実装の順番

1. 使用する写真・キャラクター画像・背景素材を決める
2. 飛行機の上下移動と背景スクロールを作る
3. アイテム配置、取得判定、カード表示を作る
4. 到着シーンとコレクション一覧を作る
5. エンディングポスターとスタッフロールを作る
6. BGMと効果音を追加する
7. PC・スマホで通し確認する
